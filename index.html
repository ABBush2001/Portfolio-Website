About Me

SAMSAT Chatbot AI

While interning at the San Antonio Museum of Science And Technology, I worked on developing a "chatbot" exhibit, in which two animatronic heads, each representing an Artificial Intelligence, held a conversation with each other. The goal of this exhibit was to provide a visual display of Artificial Intelligence to museum goers, and thanks to this project I gained experience working with Raspberry Pis, Artificial Intelligence, and Python Libraries.

Ninten-Do: Desktop checklist application

Developed in Java as part of UTSA's Application Programming Course, Ninten-Do is a desktop application that helps users manage various daily tasks in a fun, game-like style. Users can acquire "points," defeat "bosses" and increase their "level" by completing various tasks. As part of the group, my responsibilities included developing the "Stats" screen, in which users could view information about their account, as well as developing the UML diagram that represented the application. See the repository for this project here.
